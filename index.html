Scope Concept Task Reference Active Voice Cognitive Load
Scope

This style guide applies to all technical writers, engineers, subject matter experts (SMEs), and stakeholders.

  • Prevents information overload
  • Supports structured documentation
  • Reduces cognitive load
Concept
DefinitionA concept explains what something is and why it matters.
PurposeProvide background knowledge before tasks.
ExamplesSystem Overview, Safety
Task
DefinitionA task describes how to perform something.
CharacteristicStep-by-step procedure
ExampleFilter replacement procedure
Reference
DefinitionProvides structured factual information.
ExamplesTables, specifications, error codes
Active Voice

Use active voice to improve clarity.

Example (Good)Tighten the connector
Example (Avoid)The connector should be tightened
Cognitive Load

Cognitive load refers to the mental effort required to process information.

  • Keep sentences short
  • Use clear structure
  • Avoid unnecessary information
